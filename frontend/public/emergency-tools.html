Ferramentas de Emergência

Voltar para o Sistema

⚠️ ATENÇÃO

Estas ferramentas são para uso em situações de emergência e podem causar perda de dados se usadas incorretamente.

Use apenas quando necessário e com extrema cautela.

Ferramentas Disponíveis

Correção de Emergência

Ferramenta principal para correções de emergência no sistema. Permite aplicar correções básicas, restaurar dados completos, limpar o sistema e mais.

Acessar

Limpar Apenas Usuários

Remove todos os usuários do sistema, exceto o administrador. Útil quando há problemas com usuários específicos.

Acessar

Redefinir Todos os Usuários

Ferramenta avançada para redefinir completamente todos os usuários do sistema, mantendo apenas o admin. Garante sincronização em todas as áreas de armazenamento.

Acessar

Redefinição Direta de Usuários

Ferramenta de emergência para redefinição direta de todos os usuários. Use esta ferramenta quando as outras opções não estiverem funcionando corretamente.

Acessar

Execução de Script de Limpeza

Ferramenta de último recurso que executa um script JavaScript diretamente para limpar todos os usuários. Use apenas se nenhuma outra opção funcionar.

Acessar

Corrigir Página de Usuários

Ferramenta para corrigir a página de usuários (/users) quando ela não estiver mostrando os dados corretos após limpar os usuários.

Acessar

Resetar Usuários (Nova Versão)

Ferramenta que utiliza o novo UserStore para resetar todos os usuários. Esta é a versão mais recente e recomendada.

Acessar

Executar Script de Reset

Ferramenta que executa diretamente um script JavaScript para resetar todos os usuários usando o novo UserStore.

Acessar

Sistema Unificado de Usuários

Ferramenta que utiliza o novo UserManager unificado para resetar todos os usuários. Esta é a versão mais avançada e recomendada.

Acessar

Informações Importantes

  • Todas as ferramentas mantêm o usuário admin com email admin@sistema.com e senha admin123
  • Após usar qualquer ferramenta, é recomendado fazer login novamente no sistema
  • Em caso de problemas, entre em contato com o suporte técnico
